# Desktop installation

1. Run `START_DESKTOP.cmd` to open BridgeDesk.
2. Run `INSTALL_DESKTOP_SHORTCUTS.cmd` to create desktop shortcuts.
3. Put tablet JSON files into `data/inbox`.
4. Run `EXPORT_HWP_REPORT.cmd` to create HWP-friendly TXT, HTML, CSV, and JSON files.

If `dist/BridgeDesk/BridgeDesk.exe` exists, shortcuts use that standalone app.
Otherwise they use the Python launcher scripts.
