# Tablet installation

1. On the desktop PC, run `START_TABLET_SERVER.cmd`.
2. Keep that window open.
3. On the tablet, connect to the same Wi-Fi network.
4. Open the URL printed by the server, usually `http://DESKTOP-IP:8765/tablet/`.
5. Use the browser menu to add the page to the home screen.
6. Enter field findings and export JSON.
7. Copy the exported JSON file into the desktop `data/inbox` folder.

The tablet page is a browser app. It does not require the App Store or Play
Store, and it can cache its files after the first successful load.
