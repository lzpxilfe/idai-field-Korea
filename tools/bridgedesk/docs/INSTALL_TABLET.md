# 태블릿 설치와 현장 입력

태블릿 앱은 앱스토어 설치 앱이 아니라 브라우저 앱입니다. 데스크톱 PC에서 작은 서버를 켜고, 태블릿 브라우저로 접속한 뒤 홈 화면에 추가해서 사용합니다.

## 태블릿 화면 열기

1. 데스크톱 PC와 태블릿을 같은 Wi-Fi에 연결합니다.
2. 데스크톱 PC에서 `START_TABLET_SERVER.cmd`를 더블클릭합니다.
3. 검은 창에 표시되는 주소를 봅니다.
   - 예: `http://192.168.0.12:8765/tablet/`
4. 태블릿 브라우저에서 그 주소를 엽니다.

`127.0.0.1` 주소는 데스크톱 PC 자기 자신을 뜻하므로 태블릿에서는 열리지 않습니다. 태블릿에서는 `192.168...`처럼 보이는 주소를 사용하세요.

## 홈 화면에 추가

### iPad 또는 iPhone

1. Safari에서 태블릿 주소를 엽니다.
2. 공유 버튼을 누릅니다.
3. `홈 화면에 추가`를 선택합니다.
4. 이름을 확인하고 추가합니다.

### Android 태블릿

1. Chrome에서 태블릿 주소를 엽니다.
2. 오른쪽 위 메뉴를 누릅니다.
3. `홈 화면에 추가` 또는 `앱 설치`를 선택합니다.
4. 이름을 확인하고 추가합니다.

## 현장 입력 후 데스크톱으로 전달

1. 조사 정보와 현장 항목을 입력합니다.
2. 필요하면 `항목 추가`를 눌러 여러 항목을 입력합니다.
3. `임시저장`을 누릅니다.
4. 작업이 끝나면 `JSON 내보내기`를 누릅니다.
5. 내려받은 JSON 파일을 메신저, 메일, 클라우드, USB, 파일 공유 중 편한 방식으로 데스크톱 PC로 옮깁니다.
6. 데스크톱의 `data\inbox` 폴더에 넣습니다.
7. 데스크톱에서 `START_DESKTOP.cmd` 또는 `EXPORT_HWP_REPORT.cmd`를 실행합니다.

## 오프라인 사용

태블릿 화면은 한 번 정상 접속하면 브라우저에 캐시됩니다. 다만 JSON 파일을 데스크톱으로 옮기려면 어떤 방식으로든 파일 전송은 필요합니다.

## 접속이 안 될 때

- 데스크톱 PC와 태블릿이 같은 Wi-Fi인지 확인합니다.
- 회사/기관 Wi-Fi가 기기 간 접속을 막는 경우 모바일 핫스팟이나 별도 공유기를 사용합니다.
- Windows 방화벽이 Python 또는 BridgeDesk 접근을 물으면 허용합니다.
- `START_TABLET_SERVER.cmd` 창을 닫으면 태블릿 주소도 닫힙니다.
